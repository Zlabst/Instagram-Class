# Instagram Class
The Instagram class is an effective, simple, and easy to use Instagram API by @teh_distance

This API Includes:
- Register
- Login
- Follow
- Like
- Comment
- Report

- Proxy Checker (for each method except login)

- Grab Profile Pic
- Grab User ID
- Grab Follower Count
- Grab Following Count
- Grab Full Name
- Grab Bio
